# 二叉树的遍历

## 1 三种遍历方式的非递归写法

### 1.1 先序遍历
“根左右”
只用维护一个stack：

1. 初始化：将root加入stack
2. 当stack不为空时，执行循环：
	2.1 弹出栈顶结点并输出
	2.2 将刚才栈顶的右孩子入栈，将刚才栈顶的左孩子入栈

```C++
void preOrder(TreeNode *root) {
	if (root == NULL) return;
	stack<TreeNode*> myStack;
	myStack.push(root);
	while (!myStack.empty()) {
		TreeNode *top = myStack.top();
		myStack.pop();
		cout << top->val << " ";
		if (top->right != NULL)
			myStack.push(top->right);
		if (top->left != NULL)
			myStack.push(top->left);
	}
}
```
